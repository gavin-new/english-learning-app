# 词汇数据说明

## 数据来源
本APP的词汇数据按照**沪教版新课标**编写，包含以下级别：

### 小学词汇（1-6年级）
- `grade1.json` - 小学一年级词汇（约150词）
- `grade2.json` - 小学二年级词汇（约200词）
- `grade3.json` - 小学三年级词汇（约200词）
- `grade4.json` - 小学四年级词汇（约250词）
- `grade5.json` - 小学五年级词汇（约250词）
- `grade6.json` - 小学六年级词汇（约300词）

### 中学词汇
- `middle_school.json` - 中考词汇（约1600词）
- `high_school.json` - 高考词汇（约3500词）

### 大学词汇
- `cet4.json` - 四级词汇（约4500词）
- `cet6.json` - 六级词汇（约5500词）

## 数据格式
每个词汇包含以下字段：
```json
{
  "word": "单词",
  "phonetic": "音标",
  "meaning": "释义",
  "example": "例句",
  "exampleTranslation": "例句翻译",
  "level": 难度等级(1-5),
  "category": "分类",
  "grade": 年级代码,
  "prefix": "前缀(可选)",
  "suffix": "后缀(可选)",
  "root": "词根(可选)",
  "rootMeaning": "词根含义(可选)"
}
```

## 年级代码
- 1-6: 小学1-6年级
- 7: 中考词汇
- 8: 高考词汇
- 9: 四级词汇
- 10: 六级词汇

## 难度等级
- 1: 基础词汇
- 2: 初级词汇
- 3: 中级词汇
- 4: 中高级词汇
- 5: 高级词汇

## 特点
1. **完全离线** - 所有数据内置，无需网络
2. **词根词缀** - 包含词根词缀分析
3. **例句丰富** - 每个单词都有例句
4. **分级科学** - 按年级和难度分级
5. **新课标** - 符合沪教版新课标要求
