## 🔧 如何补充词汇数据

### 方法1：手动添加（推荐小批量）

1. 打开对应的JSON文件（如 `grade4.json`）
2. 按照以下格式添加词汇：

```json
[
  {
    "word": "example",
    "phonetic": "/ɪɡˈzɑːmpl/",
    "meaning": "n. 例子；榜样",
    "example": "This is an example.",
    "exampleTranslation": "这是一个例子。",
    "level": 2,
    "category": "小学四年级",
    "grade": 4
  }
]
```

### 方法2：批量导入（推荐大批量）

1. 准备CSV文件（格式：单词,音标,释义,例句,例句翻译）
2. 使用在线工具转换为JSON
3. 复制到对应的JSON文件
