# EnglishLearningApp - 英语学习应用

📱 一款功能全面的英语学习APP，支持背单词、听力练习、阅读理解等核心功能。

## ✨ 功能特色

### 📝 背单词
- 记忆卡片展示（单词、音标、例句）
- **艾宾浩斯遗忘曲线**智能复习算法
- 学习进度追踪
- 单词测试（选择、拼写、听力）

### 🎧 听力练习
- 音频播放器（支持暂停、快进、倒退）
- **听写练习**模式
- 语速调节（0.5x - 2.0x）
- 多难度级别听力材料

### 📖 阅读理解
- 文章分类阅读
- **点击查词**功能
- 生词收藏
- 阅读进度追踪

### 🔍 智能查询
- **字母查询** - 输入单词快速搜索
- **语音查询** - 说话即可查询
- **拍照查询** - 拍照识别文字
- **词根查询** - 前缀/后缀/词根分析

### 📊 学习中心
- 今日/本周/累计学习统计
- 学习目标设置
- 连续学习天数追踪

## 🛠 技术架构

### 技术栈
- **语言**: Kotlin
- **最低版本**: Android 7.0 (API 24)
- **UI框架**: Jetpack Compose
- **数据库**: Room（离线存储）
- **架构**: MVVM
- **AI功能**: Google ML Kit（拍照识别）

### 项目结构
```
com.englishlearning.app/
├── data/
│   ├── local/          # Room数据库
│   └── model/          # 数据模型
├── ui/
│   ├── home/          # 首页
│   ├── vocabulary/    # 背单词
│   ├── listening/     # 听力练习
│   ├── reading/       # 阅读理解
│   ├── search/        # 搜索功能
│   └── theme/         # 主题样式
└── utils/            # 工具类
```

## 🚀 快速开始

### 环境要求
- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34
- Gradle 8.0

### 安装步骤

#### 1️⃣ 下载项目
```bash
# 克隆项目（如果有）
git clone https://github.com/yourusername/EnglishLearningApp.git
cd EnglishLearningApp
```

#### 2️⃣ 使用Android Studio打开
1. 打开 Android Studio
2. 选择 "Open an existing project"
3. 选择项目根目录
4. 等待 Gradle 同步完成

#### 3️⃣ 运行项目
1. 连接安卓设备或启动模拟器
2. 点击 Android Studio 的绿色运行按钮 ▶️
3. 等待编译完成，应用将自动安装到设备上

#### 4️⃣ 生成APK
```bash
# 在项目根目录执行
./gradlew assembleDebug

# APK文件位置
app/build/outputs/apk/debug/app-debug.apk
```

### 适配设备
- ✅ **Android 7.0+** 的所有安卓手机
- ✅ **鸿蒙 4.0** 及以下版本的华为手机
- ⚠️ 鸿蒙 NEXT 可能需要特殊适配

## 📖 使用指南

### 背单词技巧
1. 每天学习20个新单词
2. 按照艾宾浩斯曲线复习（1天后、3天后、7天后...）
3. 查看单词的词根词缀，加深理解
4. 收藏不认识的单词到生词本

### 听力练习技巧
1. 先听2-3遍，理解大意
2. 尝试听写，填写听到的内容
3. 查看原文，对照学习
4. 听不懂的单词要查词典

### 阅读理解技巧
1. 先看标题，预测内容
2. 阅读时遇到生词，点击查看释义
3. 遇到重要单词，收藏到生词本
4. 读完后回顾总结

### 智能查询使用
- **字母查询**: 输入完整或部分单词
- **语音查询**: 点击麦克风，说出单词
- **拍照查询**: 对准文字拍照，自动识别
- **词根查询**: 输入词根（un-, -tion, re-等）

## 🔧 自定义配置

### 修改每日学习目标
在 `HomeScreen.kt` 中修改目标数量

### 添加新单词
在 `AppDatabase.kt` 的 `populateDatabase` 方法中添加

### 调整艾宾浩斯复习间隔
在 `LearningProgress.kt` 中修改 `REVIEW_INTERVALS`

## 🐛 常见问题

### Q: 应用无法安装？
A: 检查手机系统版本是否在 Android 7.0 以上

### Q: 录音功能不工作？
A: 前往设置 → 应用 → 权限，确保授予麦克风权限

### Q: 拍照识别失败？
A: 确保光线充足，文字清晰可见

### Q: 华为手机无法安装？
A: 如果是鸿蒙 NEXT 版本，可能需要应用市场审核版
