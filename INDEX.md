## 🛠 如何使用

### 方式1: Android Studio
```
打开Android Studio
→ Open existing project
→ 选择 EnglishLearningApp 文件夹
→ 等待同步完成
→ Run (绿色三角)
```

### 方式2: 命令行
```bash
gradlew assembleDebug
# APK位置: app\build\outputs\apk\debug\app-debug.apk
```

---

## 📱 安装APK
```
将 app-debug.apk 复制到手机
→ 点击安装
→ 开启"未知来源"（如提示）
→ 启动应用
```
