# EnglishLearningApp - 英语学习APP设计方案

## 📱 项目概述

**项目名称**: EnglishLearningApp  
**目标用户**: 英语学习者（初学者到中级）  
**核心功能**: 背单词 + 听力练习 + 阅读理解

## 🎯 功能模块设计

### 1. 背单词模块 📝
- **记忆卡片**: 展示单词、音标、例句
- **艾宾浩斯复习**: 基于遗忘曲线自动安排复习时间
- **学习进度**: 显示已掌握/学习中/未学习的单词
- **单词测试**: 多种题型（选择、拼写、听力）

### 2. 听力练习模块 🎧
- **音频播放器**: 播放听力材料，支持暂停/快进/倒退
- **听写练习**: 播放句子，填写听到的内容
- **语速调节**: 0.5x - 2.0x 变速播放
- **听力材料库**: 内置多级别听力材料

### 3. 阅读理解模块 📖
- **文章列表**: 按难度分类的文章
- **阅读界面**: 流畅的阅读体验
- **词汇点击**: 点击单词查看释义
- **生词收藏**: 一键收藏不认识的单词

### 4. 学习中心 📊
- **学习统计**: 今日/本周/累计学习数据
- **学习目标**: 设置每日学习目标
- **学习提醒**: 定时提醒功能

## 🛠 技术架构

### 技术栈
- **语言**: Kotlin
- **最低版本**: Android 7.0 (API 24)
- **UI框架**: Jetpack Compose (现代声明式UI)
- **数据库**: Room (本地数据持久化)
- **架构模式**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **异步处理**: Kotlin Coroutines

## 🎨 UI/UX 设计

### 主题风格
- **主色调**: 蓝色 (#1976D2) - 知识、信任
- **辅助色**: 绿色 (#4CAF50) - 进步、成功
- **背景色**: 浅灰 (#FAFAFA) - 简洁舒适
- **字体**: 默认系统字体

### 导航设计
- **底部导航栏**: 4个主要入口
  1. 首页 🏠
  2. 背单词 📝
  3. 听力 🎧
  4. 阅读 📖

### 学习流程
```
首页 → 选择学习模块 → 开始学习 → 查看进度 → 复习提醒
```

## 📊 数据设计

### 数据库表结构

#### 1. words 表（单词表）
- id: 主键
- word: 单词
- phonetic: 音标
- meaning: 释义
- example: 例句
- audio_url: 音频地址
- level: 难度等级(1-5)
- is_learned: 是否已学习

#### 2. articles 表（文章表）
- id: 主键
- title: 标题
- content: 内容
- level: 难度等级
- word_count: 单词数

#### 3. learning_progress 表（学习进度表）
- id: 主键
- word_id: 单词ID
- next_review_date: 下次复习日期
- review_count: 复习次数
- mastery_level: 掌握程度(0-5)

#### 4. user_stats 表（用户统计表）
- id: 主键
- date: 日期
- words_learned: 今日学习单词数
- listening_time: 听力时长(分钟)
- reading_time: 阅读时长(分钟)
